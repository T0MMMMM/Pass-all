Aide — Pass-all

Bienvenue dans l'aide de Pass-all, votre gestionnaire de mots de passe local et sécurisé.

◆ À quoi sert Pass-all ?

Pass-all est une application qui vous permet de stocker tous vos mots de passe en un seul endroit, protégé par un mot de passe maître que vous seul connaissez.

Vous n'avez plus besoin de retenir des dizaines de mots de passe ou de les noter sur un papier. Pass-all les garde pour vous, chiffrés sur votre ordinateur — aucune donnée ne quitte votre machine.

Important : Choisissez un mot de passe maître solide et ne l'oubliez pas. Il est la seule clé pour accéder à tous vos autres mots de passe.

◆ Connexion et inscription

C'est la première page que vous voyez au lancement de l'application.

Se connecter

Si vous avez déjà un compte Pass-all sur cet ordinateur :

  1. Entrez votre identifiant dans le premier champ.
  2. Entrez votre mot de passe maître dans le second champ. Cliquez sur l'œil à droite pour l'afficher si besoin.
  3. Cliquez sur Se connecter.
Si votre identifiant ou mot de passe est incorrect, un message d'erreur s'affiche en rouge. Vérifiez vos saisies.

Créer un compte

Si c'est votre première utilisation, cliquez sur le lien « Pas encore de compte ? Créer un compte » en bas du formulaire.

  1. Entrez votre nom complet.
  2. Entrez votre adresse e-mail.
  3. Choisissez un identifiant unique (ce sera votre nom d'utilisateur).
  4. Choisissez un mot de passe maître fort. Entrez-le une seconde fois pour confirmer.
  5. Cliquez sur Créer mon compte.
Bon à savoir : Votre mot de passe maître n'est jamais stocké en clair — il est transformé de façon irréversible avant d'être sauvegardé.

◆ Mes comptes

C'est la page principale de l'application. Elle liste tous les comptes (sites, applications, services) dont vous avez enregistré les identifiants et mots de passe.

Voir la liste de mes comptes

Chaque compte est affiché sous forme de carte avec :

  • L'initiale du service dans un cercle coloré (la couleur correspond à sa catégorie).
  • Le nom du service (ex : Gmail, Netflix…).
  • La catégorie à laquelle il appartient.
  • L'identifiant (votre nom d'utilisateur ou e-mail pour ce service).
  • Le mot de passe, masqué par défaut (affiché comme ••••••••).

Afficher ou copier un mot de passe

Afficher le mot de passe

Cliquez sur l'icône œil à droite de la carte pour révéler le mot de passe. Cliquez à nouveau pour le masquer.

Copier le mot de passe

Cliquez sur l'icône copier (deux feuilles) pour copier le mot de passe dans le presse-papier. Vous pouvez ensuite le coller directement dans le site ou l'application concernée.

Ajouter un compte

  1. Cliquez sur le bouton + Ajouter un compte en bas de la liste.
  2. Remplissez les champs : nom du service, identifiant, e-mail, URL du site et mot de passe.
  3. Choisissez une catégorie dans la liste déroulante (optionnel).
  4. Cliquez sur Enregistrer pour sauvegarder.
Cliquez sur l'œil dans le formulaire pour vérifier le mot de passe que vous saisissez avant de le sauvegarder.

Modifier un compte

  1. Cliquez sur la carte du compte que vous voulez modifier.
  2. Le formulaire de modification s'ouvre directement sous la carte.
  3. Modifiez les champs souhaités.
  4. Cliquez sur Enregistrer pour valider, ou Annuler pour ne rien changer.

Supprimer un compte

  1. Cliquez sur la carte du compte à supprimer pour ouvrir le formulaire.
  2. Cliquez sur le bouton rouge Supprimer.
Attention : La suppression est définitive. Le compte ne pourra pas être récupéré.

Rechercher et trier

Barre de recherche

Tapez le nom d'un service dans la barre « Rechercher un compte… » pour filtrer la liste en temps réel.

Trier la liste

Utilisez le menu déroulant à droite de la barre de recherche pour trier vos comptes : Nom A→Z, Récents ou Catégorie.

◆ Catégories

Les catégories vous permettent de regrouper vos comptes par thème (Réseaux sociaux, Banque, Travail, etc.) et de les identifier visuellement grâce à une couleur.

Pour accéder à la gestion des catégories, ouvrez les Paramètres (icône engrenage en bas à droite) puis cliquez sur Gérer à côté de « Catégories ».

Créer une catégorie

  1. Cliquez sur + Ajouter une catégorie.
  2. Donnez un nom à votre catégorie.
  3. Choisissez une couleur parmi les 8 disponibles en cliquant sur un cercle coloré.
  4. Cliquez sur Enregistrer.

Modifier une catégorie

  1. Cliquez sur la carte de la catégorie à modifier.
  2. Changez le nom et/ou la couleur.
  3. Cliquez sur Enregistrer.

Supprimer une catégorie

  1. Cliquez sur la carte de la catégorie à supprimer.
  2. Cliquez sur le bouton rouge Supprimer.
Supprimer une catégorie ne supprime pas les comptes qui lui étaient associés. Ils se retrouvent simplement sans catégorie.

Revenir à mes comptes

Cliquez sur la flèche ← en haut à gauche pour revenir à la liste de vos comptes.

◆ Paramètres

Accédez aux paramètres en cliquant sur l'icône engrenage en bas à droite de l'application.

Gérer les catégories

Cliquez sur Gérer pour accéder à la page de gestion des catégories (voir section Catégories ci-dessus).

Importer un dictionnaire

Cette fonctionnalité vous permet d'importer une liste de mots depuis un fichier CSV ou TXT. Ces mots sont enregistrés dans votre base de données locale et peuvent servir à générer des mots de passe mémorisables.

  1. Cliquez sur Importer à côté de « Dictionnaire ».
  2. Sélectionnez un fichier .csv ou .txt sur votre ordinateur. Chaque mot doit être sur une ligne séparée.
  3. Une coche verte s'affiche brièvement pour confirmer que l'import s'est bien déroulé.
Seuls les mots contenant des lettres, chiffres, espaces, tirets ou apostrophes sont importés. Les autres sont ignorés automatiquement.

Se déconnecter

Cliquez sur Se déconnecter pour fermer votre session et revenir à l'écran de connexion. Vos données restent bien sûr sauvegardées.

◆ Sécurité et confidentialité

Stockage local uniquement

Toutes vos données (comptes, mots de passe, catégories) sont stockées sur votre ordinateur, dans un fichier de base de données local. Rien n'est envoyé sur Internet.

Mots de passe chiffrés

Vos mots de passe ne sont jamais stockés en clair. Ils sont chiffrés avant d'être sauvegardés, et déchiffrés uniquement au moment où vous en avez besoin.

Mot de passe maître

Votre mot de passe maître est converti en empreinte numérique irréversible (hachage). Même en accédant directement à la base de données, il est impossible de le retrouver.

Conseil : Verrouillez votre session Windows/macOS/Linux lorsque vous vous absentez de votre ordinateur pour protéger l'accès à l'application.
